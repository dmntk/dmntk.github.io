# Decision Model and Notation Toolkit

**DMNTK** is a platform for building, testing and evaluating decision models.

**DMNTK** is based on the [Decision Model and Notation (DMN™)](https://www.omg.org/dmn/),
the industry standard led by the [Object Management Group (OMG®)](https://www.omg.org/),
the institution behind such standards like UML®, BPMN™ and CORBA® to name a few.

**DMNTK** is written in [Rust](https://www.rust-lang.org/), a programming language that empowers building
reliable and efficient software. **DMNTK** aims to be a reliable and efficient solution for executing
decision models, fully compliant with [DMN™ specification](https://www.omg.org/spec/DMN).
